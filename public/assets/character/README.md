# キャラクターアセット

## 動画（MP4）

ノアの移動・エモート・アニメーション用。`src/constants/index.ts` の `CHARACTER_VIDEOS` および `getCharacterVideoUrl()` で参照できます。

| 用途       | ファイル名        | 説明           |
|------------|-------------------|----------------|
| 移動       | `movement_01.mp4` | シーン移動時など |
| エモート   | `emote_01.mp4`, `emote_02.mp4` | 感情表現・リアクション |
| アニメーション | `animation_01.mp4` | 汎用アニメーション |

## キャラクター画像

### tyoitetu/ — TYOITETU

キャラクター「TYOITETU」の画像。`tyoitetu_01.png` ～ `tyoitetu_08.png`。`TYOITETU_IMAGES` と `getTyoitetuImageUrl(index)` で参照できます。

追加する場合は連番で配置し、定数にファイル名を追加してください。
